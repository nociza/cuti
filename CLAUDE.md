## Development Commands

### Setup and Installation
```bash

python run.py setup

# Development installation with uv
uv install -e .
```

### Running the Application
```bash
# Start web interface
python run.py web

# Start CLI
python run.py cli

# Check agent status
cuti agent list
```

3. Modify `.cuti/agents.json` and reload
