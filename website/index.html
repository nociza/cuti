Cuti

An instant, containerized Claude Code dev environment

One command for a ready-to-use, isolated Claude Code workspace in Docker

Get Started Documentation

🚀 Quick Start

Terminal 
# Install Python package from PyPI
uv tool install cuti

# Launch Docker development environment with Claude
cuti container

That's it! You now have a fully configured AI-powered coding environment.

✨ Key Features

Containerized Development

Claude Code CLI pre-configured with Anthropic integration in an isolated Docker environment

Provider Management

Manage agent-CLI providers (Claude, Codex, OpenCode, OpenClaw) and switch between multiple Claude accounts

Secure by Default

The host Docker socket and inactive-account credentials stay off-limits to the agent unless you explicitly opt in

Read-only Ops Console

Launch cuti web to inspect provider readiness, Claude sessions, and workspace drift

Persistent Authentication

Claude API authentication persists across sessions with easy account switching

History & Usage

Browse and resume past Claude Code sessions with cuti history and track local usage analytics

📦 What's Included

Claude Code CLI pre-configured with Anthropic integration

Python 3.11, Node.js 20, and essential dev tools
